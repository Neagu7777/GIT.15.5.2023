1. Как скопировать на свой компьютер репозиторий из github.com, что должно быть сделано предварительно?
bash git clone <URL репозитория>

2.Как получить на локальный компьютер изменения с github.com
bash git pull

3.Какую информацию нам дают команда :
git status
git log
git remote -v 

команды показывает состояние вашего рабочего каталога и индекса, есть ли изменения, 
которые не были зафиксированы, и есть изменения, которые не были отправлены в удаленный репозиторий.
